Mozilla Research

Pushing the boundaries of the Open Web

  • Emscripten

    Emscripten is an optimizing compiler that ports desktop applications written in C or C++ to web applications using pure HTML and JavaScript. Emscripten generates extremely efficient code and is enabling new kinds of web applications.

  • Rust

    Rust is a new systems programming language for building high-performance, concurrent software. Rust is a foundational component of the Servo project to rethink the architecture of a web browser.

  • Servo

    The Servo project aims to build a web browser engine designed for parallelism and power efficiency. Servo builds on top of Rust for better security, robustness, and concurrency.
