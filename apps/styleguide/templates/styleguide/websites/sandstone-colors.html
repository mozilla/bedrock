Backgrounds

These are the default colors and gradients for Mozilla and Firefox sites, along with a few alternates. They are guidelines only, not absolute rules. You can create different color variations within Sandstone as your project or site warrants.

The gradients shown are layered behind a default grain texture, which you can download here.

Default

Mozilla

#D7D3C8 to #F6F4EC

Firefox

#CAE1F4 to #EEEEEE

Link colors

The default colors for links in sandstone use Firefox light blue by default and Firefox dark blue for hover states. Similar to above, these can change based on custom style and should be complimentary to the chosen background colors. The link hover state should also introduce an undeline of the same color.
