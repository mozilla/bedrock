{{ _('What’s the difference between 32-bit and 64-bit?') }}

{% trans ASLR='https://en.wikipedia.org/wiki/Address_space_layout_randomization' %} Here’s the key thing to know: 64-bit applications can access more memory and are less likely to crash than 32-bit applications. Also, with the jump from 32 to 64 bits, a security feature called Address Space Layout Randomization (ASLR) works better to protect you from attackers. Linux and macOS users, fret not, you already enjoy a Firefox that’s optimized for 64-bit. {% endtrans %}
