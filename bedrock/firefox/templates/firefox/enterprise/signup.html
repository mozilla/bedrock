Mozilla allows organizations to choose from two Firefox release channels:

  • Rapid Release (RR)
    • Recommended for most organizations, especially those using SaaS web apps.
    • Auto-updates with performance enhancements and new developer features every six weeks.
    • Critical security updates ASAP.
  • Extended Support Release (ESR)
    • Recommended for organizations that extensively test each browser release against internally-developed web apps.
    • One big update annually, including all performance improvements and developer features from Rapid Release.
    • Critical security updates ASAP.
