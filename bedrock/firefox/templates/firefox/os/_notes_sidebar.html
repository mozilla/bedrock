{{ _('How do the version numbers work?') }}

{% trans %} Firefox OS has a 4 digit version number: X.X.X.Y. The first 2 digits are owned by the Mozilla product team and will denote versions with new features (eg: v1.1, 1.2, etc). The third digit will be incremented with regular version tags (~every 6 weeks) for security updates, and the fourth will be OEM owned. {% endtrans %}
