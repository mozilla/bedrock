{{ _('A Healthy Internet is Inclusive to All') }}

{% trans %} The more voices, perspective, languages, and people contributing to the Web, the richer the experience for everyone. But the whole Internet is not yet accessible, welcoming, and safe for all. Together, we have the power to shape the Web, and our world along with it. {% endtrans %} {% trans %} A healthy Internet is open to you. {% endtrans %}

{% trans %} Here are three areas where we can make a difference. {% endtrans %}

  • {{ _('Promote Diversity') }}
  • {{ _('Practice Respect') }}
  • {{ _('Support Universal Access') }}

{{ _('Digital Inclusion means promoting diversity') }}

{% trans %} For the Internet to fulfill its greatest promise, it must reflect the diversity and experience of all people, everywhere. {% endtrans %}

{% trans %} As inclusive as the Web can seem, it’s not yet an equal playing field. More than half the world is still without it; emerging economies and marginalized communities are often the last to gain access. Far fewer women are using the Internet than men. And without diversity among its creators, the Web itself will reflect unconscious biases, while personalizing algorithms can reinforce our own. {% endtrans %}

{% trans %} In nine developing countries, women are still nearly 50% less likely to access the Internet than men. {% endtrans %}

{% trans wwwf='http://webfoundation.org/about/research/womens-rights-online-2015/' %} Source: World Wide Web Foundation, 2015 {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('50%') }}

{{ _('How you can promote diversity') }}

  • {{ _('Create web content') }}

    {% trans %} Create your own Web content – like videos, blog posts, GIFs – in your language, local to your area. {% endtrans %}

  • {{ _('Build resources') }}

    {% trans wiki_support='https://donate.wikimedia.org', wiki_contribute='https://www.wikipedia.org/' %} Support a resource like Wikipedia that drastically lowers the barriers to knowledge – or better yet, help build it. Wikipedia needs more, and more diverse, contributors. {% endtrans %}

{{ _('How Mozilla is promoting diversity') }}

  • {{ _('Advancing accessibility') }}

    {% trans web_a11y='https://developer.mozilla.org/docs/Learn/Accessibility' %} Advancing web accessibility standards, and making Firefox inclusive through efforts like our Accessibility team. {% endtrans %}

  • {{ _('Building digital skills') }}

    {% trans un_women='https://learning.mozilla.org/blog/new-partnership-with-un-women-to-teach-key-digital-skills-to-women' %} Partnering with UN Women to help women in Africa and across the globe build key digital skills. {% endtrans %}

{# end #diversity #}

{{ _('Digital Inclusion means practicing respect') }}

{% trans %} We should all have the ability to participate fully on the Internet, without threat to our reputations, our confidence, or our safety. {% endtrans %}

{% trans %} We’ve all seen our share of nasty comments sections. At times, the Web can feel like a very unfriendly place – particularly for women, minorities, and members of marginalized communities. By discouraging people from getting online, cyberbullying and cyber violence threaten not just individuals, but the Internet itself. {% endtrans %}

{% trans %} 73% of Internet users have seen someone harassed online and 40% have personally experienced it. {% endtrans %}

{% trans pew='http://www.pewinternet.org/2014/10/22/online-harassment/' %} Source: Pew Research, 2014 {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('73%') }}

{{ _('How you can practice respect') }}

  • {{ _('Report bullying') }}

    {% trans report='http://cyberbullying.org/advice-for-adult-victims-of-cyberbullying' %} If you see cyber violence and bullying online, record it and report it. {% endtrans %}

  • {{ _('Talk about respect') }}

    {% trans kids='http://www.connectsafely.org/tips-to-help-stop-cyberbullying/', teachers='https://thimbleprojects.org/amira/33977/' %} Talk to your kids about cyberbullying, and encourage their teachers to do the same. {% endtrans %}

{{ _('How Mozilla is practicing respect') }}

  • {{ _('Supporting tools for safety') }}

    {% trans coral='https://coralproject.net/', talk='https://coralproject.net/products/talk.html' %} Supporting the Coral Project, which works to improve how journalists and communities engage on news websites, including tools like Talk, that aims to create smarter, safer comments sections. {% endtrans %}

  • {{ _('Empowering individuals') }}

    {% trans teaching='https://thimbleprojects.org/amira/33977/' %} Creating open curricula that empowers people to create safe spaces online, like the Teaching Kit: Combating Cyber Violence Against Women and Girls. {% endtrans %}

{# end #respect #}

{{ _('Digital Inclusion means supporting universal access to the whole Internet') }}

{% trans %} We should all have affordable, high-quality, unrestricted access to the whole Web, so the whole world can benefit. {% endtrans %}

{% trans %} To participate online, you have to be able to get online. Programs have emerged that offer free or subsidized Internet access, but it is often slow or restricted, creating a ‘poor Internet for poor people.’ At its most extreme, governments worldwide are turning off all or parts of the Internet to serve their own agendas, which can threaten human rights and even the health of the global economy. {% endtrans %}

{% trans %} In 2016, there were 51 intentional Internet shutdowns in 18 countries. {% endtrans %}

{% trans accessnow='https://www.accessnow.org/keepiton/' %} Source: AccessNow, 2016 {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('51') }}

{{ _('How you can support universal access') }}

  • {{ _('Donate your tech') }}

    {% if LANG.startswith('en-') %} Donate your old computers, laptops, and phones to non-profits like Reconnect, Students Recycling Used Technology, or Interconnection, who refurbish and redistribute them to underserved communities. {% elif l10n_has_tag('inclusion-donate-tech') %} {% trans %} Donate your old computers, laptops, and phones to non-profits that refurbish and redistribute them to underserved communities. {% endtrans %} {% else %} {% trans reconnect='http://www.dell.com/learn/us/en/uscorp1/corp-comm/us-goodwill-reconnect', strut='https://en.wikipedia.org/wiki/StRUT', interconnection='http://interconnection.org/' %} Donate your old computers, laptops, and phones to non-profits like Reconnect, Students Recycling Used Technology, or Interconnection, who refurbish and redistribute them to underserved communities. {% endtrans %} {% endif %}

  • {{ _('Advocate for openness') }}

    {% trans %} Tell your representatives that open and affordable access to the Internet should be a policy priority. {% endtrans %}

{{ _('How Mozilla is supporting universal access') }}

  • {{ _('Advocating for neutrality') }}

    {% trans neutrality='https://blog.mozilla.org/netpolicy/category/net-neutrality/' %} Championing net neutrality, so that everyone has access to the same Internet. {% endtrans %}

  • {{ _('Connecting the unconnected') }}

    {% trans equalrating='https://equalrating.com/' %} Sponsoring the Equal Rating Innovation Challenge to find novel solutions for connecting the unconnected to the full, open Internet. {% endtrans %}

{# end #access #}
