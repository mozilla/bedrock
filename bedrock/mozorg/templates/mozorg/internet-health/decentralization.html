{{ _('A healthy Internet is created and owned by us all') }}

{% trans %} The Internet owes much of its success to openness: its open, shared structure has made it easy for everyone to build, surf, and thrive on it. But a few big companies are closing in, closing doors, and creating walled gardens that concentrate their ownership and control of the Web. Together, we can fight to make sure no one limits our Internet access, experience, or creation. {% endtrans %} {% trans %} A healthy Internet belongs to you. {% endtrans %}

{% trans %} Here are four areas where we can make a difference. {% endtrans %}

  • {{ _('Net Neutrality') }}
  • {{ _('Interoperability') }}
  • {{ _('Competition and Choice') }}
  • {{ _('Local Contribution') }}

{{ _('Decentralization means Net Neutrality') }}

{% trans %} No one should be able to restrict our access to the Web for their own gain. {% endtrans %}

{% trans %} We rely on network providers – telcos and cable companies – for access to the Internet. Which puts them in a position to restrict that access for their own business objectives, favoring their own products, blocking sites or brands, or charging different prices and offering different speeds depending on content type. Net neutrality prohibits network providers from discriminating based on content, so everyone has equal access. {% endtrans %}

{% trans %} As of December 2016, 47 countries currently have laws in place to protect Net Neutrality. {% endtrans %}

{% if l10n_has_tag('health-decentralization-link') %} {% trans accessnow='https://www.thisisnetneutrality.org/' %} Source: Access Now {% endtrans %} {% else %} {% trans wwwf='https://www.thisisnetneutrality.org/' %} Source: Access Now, 2015 {% endtrans %} {% endif %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('47') }}

{{ _('How you can support net neutrality') }}

  • {{ _('Keep speaking up') }}

    {% trans %} The fight for net neutrality continues all over the world. The rules that have been adopted in the US and Europe need to be defended and enforced. Reach out to your government official wherever you live, and express your support. {% endtrans %}

{{ _('How Mozilla is supporting net neutrality') }}

  • {{ _('Advocating policy change') }}

    {% trans usa='https://blog.mozilla.org/netpolicy/2017/01/20/calling-on-the-new-administration-to-make-the-internet-a-priority/', europe='https://blog.mozilla.org/netpolicy/2016/06/20/eu-users-can-stand-up-for-net-neutrality/', india='https://blog.mozilla.org/netpolicy/2016/07/01/protecting-the-open-internet-in-india/' %} Working directly with legislative bodies to craft policy frameworks for and meaningful enforcement of net neutrality in the United States, Europe, India, and all over the world. {% endtrans %}

  • {{ _('Building community') }}

    {% trans %} Fostering a global community of passionate Internet users who share our vision. {% endtrans %}

{# end #neutrality #}

{{ _('Decentralization means Interoperability') }}

{% trans %} The Web should remain open and interoperable, so we can keep our experience consistent, transparent, and full of possibility. {% endtrans %}

{% trans %} Interoperability is a big word with a simple result: your Web experience is basically the same across browsers, hardware, and operating systems because it was designed that way – and built with the open standards to support it. Open standards also allow anyone to invent new ways to make your Web experience better. But interoperability is losing ground to closed systems – and we’re losing transparency, participation, and innovation along with it. {% endtrans %}

{% if l10n_has_tag('iot_interoperable_quote') %}

{% trans %} Nearly half the economic potential of the Internet of Things relies on its systems being interoperable. {% endtrans %}

{% trans mckinsey='http://www.mckinsey.com/business-functions/digital-mckinsey/our-insights/the-internet-of-things-the-value-of-digitizing-the-physical-world' %} Source: McKinsey, 2015 {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('50%') }}
{% endif %}

{{ _('How you can support interoperability') }}

  • {{ _('Go independent') }}

    {% trans %} Innovation can still come from anywhere – especially if we support it. Try out apps and products from companies you don’t already know. {% endtrans %}

{{ _('How Mozilla is supporting interoperability') }}

  • {{ _('Setting the standards') }}

    {% trans ietf='https://www.ietf.org/', w3c='https://www.w3.org/' %} Working with and even leading open standards bodies, like IETF and W3C. {% endtrans %}

  • {{ _('Walking the talk') }}

    {% trans mdn='https://developer.mozilla.org' %} Building interoperability into our own products, and empowering web developers through initiatives like the Mozilla Developer Network. {% endtrans %}

{# end #interoperability #}

{{ _('Decentralization means Competition and Choice') }}

{% trans %} The Internet should continue to foster healthy competition among companies, opportunity for entrepreneurs, and meaningful choices for users. {% endtrans %}

{# L10n: "the whole enchilada" is American slang for a complete package. You can substitute a similar colloquial term or translate as "the complete package" #} {% trans %} A personalized Internet is an exciting prospect. But more and more, that means opting into a single company’s ecosystem – which streamlines your experience right now, but may seriously limit your choices in the future. Competitors will be reduced to those few companies who can offer the whole enchilada, thus consolidating the power of existing tech giants and making it much harder for entrepreneurs to disrupt the market with great ideas. {% endtrans %}

{% trans %} In most EU member countries, Google controls more than 90 percent of the search market. In some countries, it controls as much as 97%. {% endtrans %}

{% trans atlantic='https://www.theatlantic.com/technology/archive/2015/04/europeans-use-google-way-way-more-than-americans-do/390612/' %} Source: The Atlantic, 2015 {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('97%') }}

{{ _('How you can influence competition and choice') }}

  • {{ _('Know the tradeoffs') }}

    {% trans %} Make sure you understand the tradeoffs of that seamless online experience before you opt into a single ecosystem. Support the companies and services that best reflect your needs – and your values. {% endtrans %}

{{ _('How Mozilla is influencing competition and choice') }}

  • {{ _('Opposing gatekeeper power') }}

    {% trans netpolicy='https://blog.mozilla.org/netpolicy/' %} Advocating for net neutrality, copyright reform, and other issues all over the world. {% endtrans %}

  • {{ _('Championing openness') }}

    {% trans standards='https://wiki.mozilla.org/Standards', mpl='https://www.mozilla.org/MPL/license-policy/' %} Building with open standards and open source in our own products, and supporting openness at every opportunity. {% endtrans %}

{# end #choice #}

{{ _('Decentralization means Local Contribution') }}

{% trans %} We should all be able to contribute to the Web, so it reflects and serves all of its users. {% endtrans %}

{% trans %} Today 3 billion people all over the world use the Internet to learn, work, play, and connect. But not everyone is able to contribute to it equally. Which means the Web doesn’t reflect the full diversity of its users, doesn’t work as well for some people as others, and can even marginalize certain communities and individuals. {% endtrans %}

{% trans %} Chinese, Spanish, Arabic and Portuguese speaking internet users make up 37.5% of the total online population, but only 11% of the Web is in their language. {% endtrans %}

{% trans wikipedia='https://en.wikipedia.org/wiki/Languages_used_on_the_Internet' %} Source: Wikipedia {% endtrans %}

{# L10n: this is used as a decorative element in a callout box. #} {{ _('37.5%') }}

{{ _('How you can support local contribution') }}

  • {{ _('Start making') }}

    {% trans thimble='https://thimble.mozilla.org/' %} Try your hand at creating Web content you care about, in your language. Thimble is a great way to start. {% endtrans %}

{{ _('How Mozilla is supporting local contribution') }}

  • {{ _('Teaching makers') }}

    {% trans tools='https://learning.mozilla.org/tools', teaching='https://learning.mozilla.org/activities' %} Providing the tools and teaching to foster the next generation of Web creators. {% endtrans %}

  • {{ _('Teaching localization') }}

    {% trans l10n='https://developer.mozilla.org/docs/Mozilla/Localization' %} Providing the blueprint for localization, so Web content can be made relevant for people in different languages and locales. {% endtrans %}
