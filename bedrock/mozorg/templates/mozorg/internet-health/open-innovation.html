{{ _('A healthy Internet is open for innovation') }}

{% trans %} The Internet was built on the promise that any one of us might create the next big thing. But in order to keep creating, imagining, and reinventing our future online, the building blocks of the Web must be open to all. And together, we need to make sure the policies and laws that govern those building blocks are fair and functional. {% endtrans %} {{ _('A healthy Internet is created by you.') }}

  • {{ _('Engage in Copyright Reform') }}
  • {{ _('Influence Patent Policy') }}

{{ _('Open Source: Keeping the building blocks of the Web open for creativity') }}

{% trans %} Open source should continue to be the heart of the Internet, so we can all see, verify, and contribute to its future. {% endtrans %}

{% trans %} Open source software – technology built with code that is open for view, use, and modification – is the engine that powers a huge amount of the Internet, from servers to operating systems to the bots that fetch your search results. It’s the infrastructure that makes the Web a truly public resource: transparent, trustworthy, and collaborative, so that anyone with an idea can contribute. But much like our IRL infrastructure, we have to commit the attention and resources to maintain it. {% endtrans %}

{% trans %} GitHub, the world’s largest open source community, includes 19 million developers, 100,000 teams, and 52 million repositories for established and developing technologies. {% endtrans %}

{% trans github='https://github.com' %} Source: GitHub {% endtrans %}

{{ _('Copyright: Reforming laws for our digital reality') }}

{% trans %} We should all be able to share, reuse and reinterpret the creativity that inspires us on the Internet. {% endtrans %}

{% trans %} Intellectual Property (IP) rights were designed as a practical solution to advance creativity in all its forms: arts, science, business, and technology. In order for IP to continue sowing seeds for the future of the Web, our laws need to support the speed and spirit of collaboration that defines our digital world. {% endtrans %}

{% trans %} There are now an estimated 1 billion Creative Commons-licensed works online that encourage reuse, including texts, photos, and music. {% endtrans %}

{% trans cc='https://stateof.creativecommons.org/2015/' %} Source: Creative Commons, 2015 {% endtrans %}

{{ _('Patents: Building a system that encourages innovation') }}

{% trans %} Our patent system should work with and for open innovation, not against it. {% endtrans %}

{% trans %} Patents were designed to create incentive for innovation. But the patent system and software development don’t always get along so well. Software patents are often written so broadly that they’re open to misinterpretation, and exclusive rights can far outlive the shelf life of the software itself. All of which creates obstacles and uncertainty for innovators, and leaves the door open for patent trolls and endless litigation. {% endtrans %}

{% trans %} High Tech patents are the subject of more disputes than all other patents combined. {% endtrans %}

{% trans patents='https://www.unifiedpatents.com/news/2016/12/28/2016-annual-patent-dispute-report' %} Source: Unified Patents, 2016 {% endtrans %}
