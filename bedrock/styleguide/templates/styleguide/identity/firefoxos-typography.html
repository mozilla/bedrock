Typeface

An open font with 897 characters (and therefore enormous localization potential), Open Sans is our default typeface for websites, print and other materials.

The Open Sans type family is used in the four weights shown here for all our communications. Open Sans Extrabold Italic in all caps is used as our display type to capture our bold voice and the spirit of the Firefox OS brand (with approved headlines only).

International typography basics

Diacritics
To accommodate diacritical marks in international languages, headlines should use one of two choices of line spacing.

Headlines with diacritics above letters should set line spacing at 100%. Headlines with diacritics both above and below letters should set line spacing at 120%.

Always follow common design sense when adjusting the leading on headlines. Diacritical marks should never interfere with one another.

  • Headlines with diacritics above
  • Open Sans Extrabold Italic
  • Size: 80pt
  • Line spacing: 100% of font size
  • Tracking: -25
  • Headlines with diacritics above and below
  • Open Sans Extrabold Italic
  • Size: 80pt
  • Line spacing: 120% of font size
  • Tracking: -25

Typography don’ts

Don't use upper and lowercase text for extrabold italic headlines

Don’t condense or scale the type disproportionately

Don't use the Extrabold capitalized italics treament for off-brand headlines.

Don’t use the wrong font

Don’t extrude the type

Don’t use excessive tracking or leading

Don’t make headlines longer than 4 lines

Don’t blur or add a drop shadow to the text
